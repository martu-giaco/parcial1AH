Cápsulas Máquinas Usuarios

Nespresso API

Esta API pensada para gestionar cápsulas de café, máquinas y usuarios de Nespresso permite listar, buscar, agregar, editar y eliminar datos de manera sencilla.

Cápsulas

GET /api/capsulas

Listar todas las cápsulas.

GET /api/capsulas/:id

Buscar cápsula por ID.

GET /api/capsulas/nombre/:nombre

Buscar cápsula por nombre.

POST /api/capsulas

Crear una nueva cápsula.

PUT /api/capsulas/:id

Actualizar una cápsula existente por ID.

DELETE /api/capsulas/:id

Eliminar una cápsula por ID.
